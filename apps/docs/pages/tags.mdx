# Tags

The routes will be used to fetch or create tags.

## Fetching all tags

`GET /tags`

Response

```json
[
  {
    "name": "Tag 1",
    "id": 1
  }
]
```

The key can be used for react keys.

## Creating a category

`POST /tags/create`

Request

```json
{
  "name": "Tag"
}
```

Response
```
Category created successfully
```
