# Job Post

This api route is to perform CR operations on job posts. It is used to create, and get job posts.

## Create Job Post

`POST /job-post`

Request Body

```json
{
  "title": "title",
  "description": "description",
  "price": 100, // optional, in USD,
  "tags": ["tag id 1", "tag id 2", "tag id 3"], // max 5,
  "category": "category id",
  "images": ["img 1", "img 2", "img 3"], // url returned from upload image api
  "deadline": "2021-01-01T00:00:00.000Z" // optional, ISO 8601 format
}
```

Response

```json
{
  "slug": "title-2f4gdsssd"
}
```

## Get Job Post using username and slug

`GET /job-post/:username/:slug`

Response

```json
{
  "title": "title",
  "description": "description",
  "tags": [
    {
      "id": "id",
      "slug": "slug",
      "name": "name"
    }
  ],
  "category": {
    "id": "string",
    "slug": "string",
    "name": "string"
  },
  "images": ["string"],
  "deadline": "Date",
  "id": "string",
  "slug": "string",
  "budget": "number",
  "createdAt": "Date",
  "author": {
    "name": "string",
    "verified": "boolean",
    "avatarUrl": "string",
    "username": "string",
    "profileCompleted": "boolean"
  },
  "claimed": "boolean",
  "claimedBy": {

  } // null if claimed is false
}
```
