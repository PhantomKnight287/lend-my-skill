# Static Data

These routes will be used by Nextjs for ISR

## Profiles

`GET /static/profiles`

Returns a list of usernames

Response:

```json
[
  {
    "username": "phantomknight287"
  }
]
```

## Services

`GET /static/services`

Returns a list of services

```json
[
  {
    "slug": "i-will-just-make-any-site-for-you-uwu-64SU3HZEBL",
    "user": {
      "username": "phantomknight287"
    }
  }
]
```

## Job posts

`GET /static/job-posts`

Returns a list of job posts

```json
[
  {
    "slug": "slug-wasdsdetew",
    "author": {
      "username": "phantom"
    }
  }
]
```
