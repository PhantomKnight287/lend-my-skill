Services

Services

These routes can be used to fetch and create services.

Creating a service

POST /services

Request Body

{
  "title": "Service Title",
  "description": "Service Description",
  "category": "id of category",
  "bannerImage": "banner image url",
  "images": ["image url 1", "image url 2"],
  "tags": ["tag id 1", "tag id 2"], // max 5 tags
  "features": [
    {
      "name": "Feature Name",
      "includedIn": ["package name 1", "package name 2"]
    }
  ], // has no limit
  "packages": [
    {
      "name": "Package Name",
      "price": 100, // in USD
      "description": "Package Description",
      "deliveryDays": 5 // in days
    }
  ] // no limit
}

Response

{
  "slug": "slug"
}

This slug can be used as /profile/<logged in user's username>/services/<slug>
